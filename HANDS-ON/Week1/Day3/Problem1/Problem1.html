This is a simple example of how to link an external CSS file to an HTML document.

About External CSS

External CSS allows you to separate your style rules from your HTML structure, making it easier to maintain and update your website's design. By linking an external CSS file, you can apply the same styles across multiple pages, ensuring consistency and saving time.

Benefits of Using External CSS

Using external CSS files provides several advantages:

  • Reusability: You can use the same CSS file across multiple HTML documents, which promotes consistency and reduces redundancy.
  • Maintainability: It is easier to update styles in one CSS file rather than modifying inline styles in multiple HTML files.
  • Improved Performance: Browsers can cache external CSS files, which can lead to faster load times for subsequent visits.

How to Link an External CSS File

To link an external CSS file, you need to use the <link> element within the <head> section of your HTML document. The href attribute specifies the path to your CSS file, and the rel attribute should be set to "stylesheet". Here is an example:

<link rel="stylesheet" href="styles.css">

In this example, the HTML document is linked to an external CSS file named "problem1.css". You can create this CSS file and add your styles to see how it affects the appearance of the HTML elements on the page.
